Principais formatações

Negrito / Destaque

Nesta forma temos um termo em negrito usando a tag <b> (sem semântica)

Já nesta forma temos um termo em destaque usando a tag <strong> (com semântica)

Itálico / Ênfase

Desse jeito teremos um termo itálico usando a tag <i> (sem semântica)

E desse outro jeito vamos ter um termo itálico usando a tag <em> (com semântica)

EM RESUMO

Não faz sentido usa as tags <b> e <i> no HTML5

Teste de combinação

teste Destaque + Ênfase <em><strong>TEXTO<strong><em>

Texto de marcação

Quando queremos marcar um texto como esse usamos a tag <mark>

Texto grande e pequeno

Podemos alterar o tamanho do texto deixando ele grande ou deixando ele pequeno.


usando a tag <big> para grande, e <small> para pequeno!

Deletado e Inserido

temos a opção de deletar um texto usando a tag <del>

ou deixar ele como texto inserido, que é diferente de sublinhado, pois, ele está na forma semântica, usamos então a tag <ins>

Sobrescrito

e por último mas não menos importante usamos a tag <sup> para exemplos como x20 + 8 - 74
